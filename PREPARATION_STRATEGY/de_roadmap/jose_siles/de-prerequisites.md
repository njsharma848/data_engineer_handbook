# A Gentle Reminder for Data Engineers

- Learn SQL before Snowflake.
- Learn Python before Databricks.
- Learn warehouse concepts before dbt.
- Learn Files & Formats before Data Lakes.
- Learn relational databases before NoSQL.
- Learn Batch processing before Streaming.
- Learn partitions before performance tuning.
- Learn to ask questions before blaming the data.
- Learn Linux & networking before cloud services.
- Learn business context before optimizing queries.

**Learn the fundamentals before going deep.**
