# Required Skills & Experience

- Strong AWS experience: Redshift (DISTKEY, SORTKEY, COPY, VACUUM/ANALYZE, Spectrum), 
  Athena (partition projection, cost optimization), S3 (bucket design, versioning, 
  lifecycle, encryption, event notifications), Glue Data Catalog, CloudWatch monitoring

- Apache Airflow: DAG design, task dependencies, custom operators/hooks, sensors, 
  scheduling, backfilling, XComs, SLAs, retries, monitoring & alerting

- PySpark: RDD/DataFrame APIs, UDFs, broadcast variables, performance tuning, 
  reading/writing Parquet/Avro/JSON/CSV, troubleshooting (Spark UI, logs)

- Python: Clean modular code, decorators, packaging, virtual environments, error 
  handling, logging, typing, unit testing (pytest), API integration

- SQL: Advanced SQL (joins, subqueries, window functions, CTEs, aggregations, 
  EXPLAIN/QUERY PLAN optimization, data deduplication)

- Data Modeling: Dimensional modeling (star/snowflake schemas), normalization/
  denormalization, fact and dimension tables, SCD types, metadata management

- ETL/ELT Concepts: Incremental loads, CDC, SCD types, late-arriving dimensions, 
  schema evolution, idempotency, data lineage, API ingestion patterns

- Testing/Quality: Data validation frameworks (Great Expectations), schema validation, 
  data profiling, reconciliation, synthetic test data, boundary & regression tests

- Security & Compliance: Encryption (at-rest/in-transit), IAM, data masking, PII 
  handling, access control, audit logging

- CI/CD & Operations: Git workflows, code reviews, automated testing, deployments to 
  Airflow/Composer, infrastructure-as-code, monitoring, cost optimization, troubleshooting
